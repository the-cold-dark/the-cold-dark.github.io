Genesis is the primary software package of the Cold Project. It is a stack-based byte-code ColdC interpreter, originally derived from ColdMUD. Two aspects of ColdC make it unique, first is its inherently persistent objects, and second is its Run-Time Mutability. A ColdC server allows changes to programs (recompiling) and objects--if not the entire object heirarchy--while running and interpreting the database. This model lends itself to the modern server architecture which is becoming more and more highly available, where downtimes of any sort are very undesirable.

Genesis 1

Genesis 1.1-STABLE is the current production release of the ColdC 1.0 driver. Genesis 1.2-DEV is the development release of the same driver.
